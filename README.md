# Интернет-магазин МТС

Spring Boot приложение, реализующее бизнес-процесс интернет-магазина на основе BPMN-диаграммы.

## Функциональность

- Просмотр каталога товаров
- Добавление товаров в корзину
- Оформление заказа с выбором способа доставки (курьер/ПВЗ)
- Интеграция с платежной системой (заглушка)
- Отслеживание статуса заказа
- Уведомления по email (логирование)

## Технологии

- Java 17
- Spring Boot 3.2.3
- Spring Data JPA
- PostgreSQL
- Lombok
- Thymeleaf

## Запуск

1. Убедитесь, что PostgreSQL запущен и доступен по адресу `localhost:5432`
